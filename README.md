🛡️ SOC L1 Toolkit: Advanced SPL Generator

Advanced SPL Generator — це локальний веб-додаток на базі Python (Flask), розроблений для автоматизації та стандартизації роботи аналітиків першої лінії (SOC Tier 1). 

Проєкт створено в рамках виконання практичного завдання для RASP Cyber Academy (Raiffeisen Bank Ukraine).

Проблематика
Під час обробки інцидентів (Triage) аналітики L1 часто витрачають дорогоцінний час на згадування складного синтаксису Splunk SPL, що збільшує загальний показник MTTA (Mean Time to Acknowledge). Крім того, людський фактор у стресових ситуаціях призводить до синтаксичних помилок у запитах та неповної доказової бази (Evidence) при ескалації на L2.

Рішення
Цей інструмент конвертує стандартизовану методологію реагування (Triage Matrix) у готовий код. Аналітику достатньо обрати вектор атаки, ввести відомі індикатори компрометації (IOCs), і додаток миттєво згенерує синтаксично правильний, оптимізований SPL-запит.

Основні можливості (Features):
* 10 підготовлених Use Cases: Охоплює найчастіші алерти банку (Malware, Brute Force, Phishing, Impossible Travel, LotL, Data Exfiltration тощо).
* Динамічні пайплайни (Pipelines): Можливість "на льоту" додавати модифікатори: `| table`, `| stats dc()`, `| dedup`, сортування та кастомні фільтри.
* Захист від помилок (Failsafe): Якщо аналітик не знає повного значення (наприклад, точного шляху до файлу), скрипт автоматично підставляє wildcards (`*`).
* SOC-friendly UI: Темна тема, адаптована для комфортної роботи в операційному центрі.
